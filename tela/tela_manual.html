TELA

PANEL

Displays Various properties in a GUI.

BRUSH

Blend : Brush blend mode. Edit text to select the blending mode, all errors default to "Normal" mode.
Size : Size of the brush in px.
Opacity : Opacity of the brush in percentage.
Flow : Flow of the brush in percentage.
SOF : Save lock values for size, opacity and flow settings. Shift+LMB - Apply lock. Ctrl+LMB - Save current to lock. Alt+LMB - Reset to default values.

CANVAS

Panel Canvas : Displays Eraser and Document Ratio and Rotation.
Rotation : Active rotation of the canvas.
Zoom : Active zoom of the canvas.
Exposure : HDR Exposure.
Gamma : HDR Gamma.
Mirror : Check to activate mirror mode.
Wrap : Check to activate wrap mode.

DOCUMENT

Width : document width in px. Edit value to change width.
Hheight : document height in px. Edit value to change height.
Resolution : document resolution in px/inch. Edit value to change resolution.
REF : Check to set reference for current width and height of the document. When active enables document deltas and deactivating it resets to previous values.
Delta Up : document delta up.
Delta Left : document delta left.
Delta Right : document delta right.
Delta Down : document delta down.

GUIDES

Guides Horizontal List : Each list entry displays the guides location in the vertical axis. When REF is active the first two guides are the original size of the document without the deltas.
Guides Vertical List : Each list entry displays the guides location in the horizontal axis. When REF is active the first two guides are the original size of the document without the deltas.
Sort Mode : Guides list will be sorted when displayed except for REF guides.
Mirror Mode : Guides will mirror each other in pairs from top to bottom considering the REF size.
Guides Visible : Check to toggle visibility of guides.
Guides Lock : Check to toggle lock of guides.

Slider Operators

Shift+LMB - Apply lock values.
Ctrl+LMB - Pin to 10% increments.
Alt+LMB - Unit increments of minus(left) or plus(right).

SETTINGS

Preview : Adjust the time interval to refresh the display Image.
Record : Open a window that mimics Krita's display meant for recording.
