### Purpose

Tela is a Krita plugin for Quick Settings.

### Features

* Canvas Settings:
  * Rotation.
  * Zoom.
  * Mirror.
  * Wrap.
* Document:
  * Width.
  * Height.
  * Resolution.
  * Reference & Deltas.
  * Guides Horizontal.
  * Guides Vertical.
  * Sort Mode.
  * Mirror Mode.
  * Guides Visible.
  * Guides Lock.
* View:
  * Brush Blending Mode.
  * Brush Size.
  * Brush Opacity.
  * Brush Flow.
  * SOF quick save.
  * View Exposure.
  * View Gamma.
