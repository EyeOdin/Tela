### Purpose

Tela is a Krita plugin for Quick Settings.

### Features

* Panel:
  * Eraser
  * Rotation
  * Mirror  
  * Wrap
* Brush:
  * Blending Mode
  * Size
  * Opacity
  * Flow
  * SOF quick save
* Canvas:
  * Rotation
  * Zoom
  * Exposure
  * Gamma
  * Mirror
  * Wrap
* Document:
  * Width
  * Height
  * Resolution
  * Reference & Deltas
* Guides:
  * Guides Horizontal
  * Guides Vertical
  * Sort Mode
  * Mirror Mode
  * Guides Visible
  * Guides Lock
* Settings:
  * Preview Interval
  * Record Window
