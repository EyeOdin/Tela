# Purpose

Tela is a Krita plugin Canvas Tool Box.

# Features

* Compact Tool Box
* Action Options
* View Options
* MirrorFix
* Color Picker ( connects to ![Pigment.O](https://github.com/EyeOdin/Pigment.O) )
* Read ![wiki](https://github.com/EyeOdin/Tela/wiki) for more information
